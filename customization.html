Asset Pipeline

All assets are read from ./assets/data/assets-manifest.json. After dropping new files into any customization folder, run ./refresh-assets.ps1 to regenerate the manifest.

Visual Armory Overview

Three asset streams, one battlefield identity.

Each category is independently replaceable. Swap a flag folder, a rank pack, a piece silhouette set, or a board skin without touching the engine.

Country Flags

Nation banners rendered on piece badges. Folders contain a flag.png (small badge icon) and a preview.png (large showcase image). Supported aspect ratios are preserved.

Piece Design

Silhouette frames and color overlays loaded from folders. A single design folder can supply multiple frame_*.png variants and a matching preview.png for the showcase carousel.

Board Skins

Manifest-loaded visual themes for the tactical grid. Each skin folder contains a preview.png for the showcase and the actual board texture applied to the .battle-board__skin element during gameplay.
