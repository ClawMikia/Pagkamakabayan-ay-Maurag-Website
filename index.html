Refresh Workflow

Because static hosting cannot enumerate folders.

  1. Drop new PNG, JPG, WEBP, GIF, or SVG files into ./custom/country-flags, ./custom/piece-designs (or its subfolders: flag, regular, skirmish, frames), ./custom/piece-colors, ./custom/board-skins, or the other customization folders.
  2. Run powershell -ExecutionPolicy Bypass -File .\refresh-assets.ps1.
  3. The script rewrites ./assets/data/assets-manifest.json with relative URLs and metadata.
  4. The site fetches that manifest at load time and updates previews, setup selectors, saved battle cosmetics, and board presentation.
